# Landscapers Portfolio (React)

Simple React app (Vite) for a landscaping business portfolio. Features:
- Home, About, Portfolio, Contact pages
- Add projects with thumbnail and client name
- Per-project gallery (upload 1-5 images) and short description
- Upload a site background image via the header control

Quick start:

1. Install dependencies

```bash
npm install
```

2. Run dev server

```bash
npm run dev
```

Notes:
- Projects and background image are stored in `localStorage` in your browser.
Default background file:

Place your image at `public/background.jpg` (project root `public` folder). If you named the file differently (for example `backround.jpg`), the app will attempt to load that filename instead — it's best to name it `background.jpg`.

Example path on your machine:

```
C:\Users\Dilip\TRANSCAPES\public\background.jpg
```
